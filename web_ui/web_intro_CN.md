# MLA-V3 Web UI 使用指南

## 功能概览

MLA-V3 Web UI 提供了一个直观的图形界面，让您能够轻松地与 Agent 进行交互并管理任务文件。

## 界面布局

界面分为两个主要区域：

### 左侧：聊天区域
- **对话界面**：与 Agent 进行实时交互
- **消息流**：显示 Agent 的响应、工具调用和任务进度
- **输入框**：输入任务指令和回复

### 右侧：文件浏览器区域
- **文件浏览**：查看当前 workspace 下的所有文件
- **文件上传**：上传文件到当前 workspace
- **文件下载**：下载 workspace 中的文件
- **实时更新**：Agent 创建或修改的文件会实时显示

## 使用流程

### 1. 创建/选择任务

1. 在顶部输入框中输入 **Task ID**
2. 点击 **Confirm** 按钮
3. 系统会自动创建或切换到对应的 workspace

### 2. 与 Agent 交互

1. 在底部输入框中输入任务指令
2. 按 **Enter** 或点击 **Send** 发送消息
3. Agent 会开始执行任务，实时显示进度和结果
4. 在右侧文件浏览器中可以实时查看 Agent 生成的文件

### 3. Human-in-Loop 交互

当 Agent 调用 `human_in_loop` 工具时：

1. **聊天框会红色闪烁**，提示需要您的输入
2. 在输入框中输入回复内容
3. 点击 **Send** 提交回复
4. Agent 会根据您的回复继续执行任务

> 💡 **提示**：HIL 交互期间，您可以随时查看右侧文件浏览器中的文件，以便更好地理解任务进度并做出决策。

### 4. 复制任务

完成任务后，如果需要保存当前任务的副本：

1. 点击 **Copy Task** 按钮
2. 输入新的 Task ID
3. 系统会将当前 workspace 的所有文件复制到新任务

### 5. 清除任务

如果需要删除任务及其所有数据：

1. 点击 **Clear Task** 按钮
2. 系统会删除：
   - workspace 目录下的所有文件
   - 主目录下的对话历史文件（conversation files）

> ⚠️ **警告**：清除操作不可恢复，请谨慎操作。

## 注意事项

- 每个 Task ID 对应一个独立的 workspace
- 对话历史会自动保存，刷新页面后可恢复
- 文件浏览器实时同步 workspace 中的文件变化
- 支持多任务切换，只需更改 Task ID 并点击 Confirm
